# react-crash-course

Build a react todolist app as a beginer with state, effect and ref hooks.

## Version

* npm v10.2.4
* Node.js v20.11.1

## Installation

```
$ npm install
```

## Usage

```
# Run mock server
$ npm run server

# Run react app
$ npm run start
```
